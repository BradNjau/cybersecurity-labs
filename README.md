# Cybersecurity Labs 🚀

This repository documents my journey learning cybersecurity through hands-on practice.

## 🔍 Skills I'm Building
- Network Traffic Analysis (Wireshark)
- Network Scanning (Nmap)
- Linux Fundamentals
- Security Concepts

### 1. Wireshark Traffic Analysis
- Captured live network traffic

- Identified open ports and services

### 3. Linux Basics
- Practiced command line usage
- Managed file permissions

## 📌 Goal
To become a skilled cybersecurity analyst through consistent practice and real-world simulations.
